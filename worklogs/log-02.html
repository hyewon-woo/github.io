Project Proposal

Project Summary

This project is continuition of my own project "readYou" from application development course at Whitworth University. readYou is a journaling web application designed to encourage users to journal regularly to allow daily reflection time for mental health. Current version of readYou allows user to submit journal, and recieve suggested music & coloring sheets from implemented APIs. However, current version of readYou faces NLP challenges. It is unable to recognize sarcasm, dialectal or cultural expressions. Followed by its challenges with NLP, it is not able to provide fully personalized activity suggestions.

Through this Project "readYou ver.2", I am intending to upgrade readYou more user friendly, user understanding, and user helpful application. I am looking into creating mobile app version of readYou for better accessibility for users to take journal whenever and wherever. I am also hoping to intergrate NLP that is awware of sarcasm, dialects and cultural expressions to analyze journals better in order to provide fully personalized suggestions. In order to do so, Journal storage is the first thing to be nailed down, as long as frame work to develop mobile app for it.

I am planning on using either Swift or Flutter to develop mobile app for readYou ver.2. To get expert's feedback, I Will be reaching out to Sarah Immel and Caterina Moruzzi from University of Edinburgh. Furthermore, I will be inputting my own journals on regular basis to develop a small case study using real user data.
